Для використання додатку необхідно запустити його на локальному сервері. 
Після запуску додатку можна використовувати ендпоінти 
`GET /api/rhombus/area` та `GET /api/rhombus/perimeter` 
з відповідними параметрами запиту.

Приклад запиту для обчислення площі ромба з довжиною сторони 5 та кутом 60 градусів:

GET /api/rhombus/area?side_length=5&angle=60
{
    "area": 10.83
}


Приклад запиту для обчислення периметру ромба з довжиною сторони 5 та кутом 60 градусів:

GET /api/rhombus/perimeter?side_length=5&angle=60
{
    "perimeter": 20
}
